📧 Spam Detection Web App

A machine learning-based web application that classifies messages as Spam or Not Spam using Natural Language Processing (NLP) techniques and a trained classification model.

📌 Project Overview

This project builds an end-to-end spam detection system that:

Processes raw text messages

Converts them into numerical features using TF-IDF

Uses a machine learning model to classify messages

Provides real-time predictions through a web interface

🧠 Features

Text preprocessing (lowercasing, stopwords removal, stemming)

TF-IDF vectorization with n-grams

Logistic Regression model with probability calibration

Real-time spam prediction with confidence score

Interactive and user-friendly Streamlit UI

🛠️ Tech Stack

Programming Language: Python

Libraries:

Scikit-learn

NLTK

Pandas, NumPy

Frontend/UI: Streamlit

Deployment: Streamlit Cloud

📊 Model Performance

Accuracy: ~97%

Spam Precision: ~0.88

Spam Recall: ~0.94

The model maintains a good balance between detecting spam and avoiding false positives.
